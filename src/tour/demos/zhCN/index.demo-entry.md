# 浮动按钮 Float Button

跟 `Back Top` 很像，多一份颜值，多一份互动。

自 `2.38.0` 开始提供。

## 演示

```demo
basic.vue
```

## API

### FloatButton Props

| 名称   | 类型               | 默认值 | 说明                     | 版本   |
| ------ | ------------------ | ------ | ------------------------ | ------ |
| bottom | `number \| string` | `40`   | 按钮的 CSS `bottom` 属性 | 2.38.0 |

### FloatButtonGroup Props

| 名称 | 类型 | 默认值 | 说明 | 版本 |
| --- | --- | --- | --- | --- |
| bottom | `number \| string` | `undefined` | 按钮组的 CSS `bottom` 属性 | 2.38.0 |
